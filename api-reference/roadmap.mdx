The Outhire API is evolving toward a fully composable platform for AI-powered recruiting.

## Planned capabilities

### Phone screening API

Programmatically trigger and manage AI phone screens powered by Joel, Outhire's conversational AI agent. Configure screening criteria, schedule calls, and receive structured results — all via the API.

### Candidate search

Full-text and semantic search across your candidate pipeline. Filter by skills, experience, fit score, and more to surface the right candidates without manual review.
