Outhire offers bulk API endpoints for high-volume recruiting workflows, including batch candidate upload and bulk candidate analysis.

These endpoints are available on request and require separate access provisioning.

## Available bulk operations

- **Bulk candidate upload** — Import large batches of candidates with resume files in a single request
- **Bulk candidate analysis** — Trigger fit scoring and resume parsing across multiple candidates at once
